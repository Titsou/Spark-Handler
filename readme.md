## Installation

### Prerequisites

- Node.js v14 or higher
- npm v6 or higher

### Steps

1. **Clone the repository**
   ```sh
   git clone https://github.com/Titsou/Spark-Handler.git
   ```

2. **Navigate to the project directory**
    ```sh
    cd spark-handler
    ```

3. **Install dependencies**
    ```sh
    npm install
    ```

4. **Starting the bot**
    ```sh
    node .
    ```
